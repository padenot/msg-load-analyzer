Firefox real-time callback load analysis

Run Firefox with audio callback trace logging enabled: 

  MOZ_DISABLE_CONTENT_SANDBOX=1 MOZ_LOG=MSGTracing:5,raw,sync MOZ_LOG_FILE=tracing.log ./mach run

Execute your scenario, then load the file that has a non-zero size into this app, called tracing.log.child-X where X is an integer.
